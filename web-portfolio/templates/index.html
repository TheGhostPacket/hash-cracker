Original Implementation

Here's the core of my original hash cracking tool:

import hashlib

def crack_hash(target_hash, wordlist):
    with open(wordlist, 'r', encoding='utf-8', errors='ignore') as file:
        for word in file:
            word = word.strip()
            hashed_word = hashlib.sha256(word.encode()).hexdigest()
            if hashed_word == target_hash:
                print(f"[✅] Password found: {word}")
                return
    print("[❌] Password not found in wordlist.")

# Example usage
if __name__ == "__main__":
    print("Hash Cracker - SHA256")
    target_hash = input("Enter the SHA256 hash to crack: ")
    wordlist_file = input("Enter path to wordlist (e.g., rockyou.txt): ")
    crack_hash(target_hash, wordlist_file)

Security Lessons

Why Weak Passwords Fail

This tool demonstrates how quickly common passwords can be cracked. Dictionary attacks can test millions of password combinations per second against unsalted hashes.

Defense Strategies

  • Use strong passwords: 12+ characters with mixed case, numbers, symbols
  • Password salting: Prevents rainbow table attacks
  • Slow hash functions: bcrypt, scrypt, Argon2
  • Multi-factor authentication: Additional security layers

Educational Use Only

Important: This tool is designed exclusively for educational purposes. It demonstrates cybersecurity concepts and should only be used for:

  • Learning about password security
  • Testing your own systems
  • Authorized security research
  • Educational demonstrations

Unauthorized use against systems you don't own is illegal and unethical.
